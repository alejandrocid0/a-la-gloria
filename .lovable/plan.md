

# Actualizar react-router-dom a 6.31.0

## Cambio

Un solo archivo, una sola línea:

**`package.json`** — cambiar la versión de `react-router-dom`:
```
"react-router-dom": "^6.31.0"
```

## Qué pasará

1. Lovable detectará el cambio en `package.json`
2. Reinstalará las dependencias automáticamente
3. La app se reconstruirá con la versión parcheada
4. Todo seguirá funcionando exactamente igual

## Riesgo

Ninguno apreciable. Misma versión mayor, misma API, solo parche de seguridad.
